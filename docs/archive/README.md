# NULLA Docs Archive

This archive keeps historical material that is still useful for reference but should not be the main way people learn the project.

Archive categories:

- `handovers/` session and release handovers
- `audits/` dated audits, smoke packs, and one-off findings
- `install/` superseded install/onboarding docs
- `status/` superseded status summaries
- `openclaw/` dated OpenClaw phase briefs and help requests
- `mobile/` side-path mobile planning
- `strategy/` roadmap and speculative planning material
- `internal/` internal context and operator-only notes
- `proof/` older prep/checklist material that was superseded by canonical proof/runbook docs
- `pitch/` sales-oriented material that should not sit in the main docs path

Use the main [docs index](../README.md) for the current public-facing documentation path.
